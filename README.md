# 📄 文件转 PDF 工具

快速将 EPUB、PPT、PPTX 文件转换为 PDF

## 功能特性

- ✨ 支持 EPUB、PPT、PPTX 格式
- 🚀 一键转换，即时下载
- 📱 响应式设计，支持移动端
- 🔒 文件本地处理，不保存数据

## 技术栈

- Backend: Flask + LibreOffice
- Frontend: HTML + CSS + JavaScript
- Deploy: Railway

## 本地运行

\`\`\`bash
pip install -r requirements.txt
python app.py
\`\`\`

访问 http://localhost:5000
